# 文档索引

> 版本基线：2026-07-21 R5 正式发布候选版。第 101～10000 关已完成自动证据与项目负责人授权的 9 局 Codex 正常时速实测；正式发布上限已解除到第 10000 关。

## 阅读顺序与冲突优先级

1. `当前原型策划.md`：确定本轮开发范围和版本验收口径。
2. `战斗与关卡循环策划.md`、`经济与成长策划.md`、`配件与词条系统策划.md`：确定当前系统规则。
3. `UI与交互重构策划.md`：确定基地、战斗、升级和结算的界面行为。
4. `关卡调参与试玩记录.md`：只记录真实样本和未完成验证，不用推测数据覆盖实测。
5. `R2实机验收与高关采样.md`：R2 实机证据、固定构筑和后续三局制采样口径。
6. `R4敌人机制与成长曲线策划.md`、`R4确定性回放与500关验收.md`：第 101～500 关的机制、词缀、分段曲线、27 局证据与最终发布实测。
7. `R5高关战区与长期成长策划.md`、`R5确定性回放与10000关验收.md`：第 501～10000 关的八段曲线、长期构筑、经济、24 局自动证据和最终发布实测。
8. `高关内容差异化策划.md`：第 501～10000 关的地图主题、新职责敌人和独立挑战/生存行动规格。
9. `R4与R5真人验收记录表.md`：R4 三局、R5 六局的最终正常时速实测记录、问题复跑与发布结论。
9. `远期总策划.md`、`远期装备系统策划.md`：完整系统规则与内容目录；与实机平衡记录冲突时，以有证据的验收结果为准。

## 游戏策划

- `当前原型策划.md`：当前原型策划说明，记录已经落地的系统、已移除入口和短期迭代优先级。

- `配件与词条系统策划.md`：当前配件实例、主词条、副词条、强化、重铸、背包管理和推荐逻辑专项策划。
- `经济与成长策划.md`：当前资源、成长循环、已移除基地永久强化入口和后续经济设计建议。
- `UI与交互重构策划.md`：当前基地、战斗 HUD、局内升级、结算、响应式和无障碍的实现规格与验收标准。
- `R2实机验收与高关采样.md`：R2 两档实机结果、UI 注入自动化、固定构筑夹具、三局制记录表和经济调参门槛。
- `R4敌人机制与成长曲线策划.md`：R4 四段成长曲线、四类敌人机制、精英词缀首版和验收矩阵。
- `R4确定性回放与500关验收.md`：R4 四段固定构筑、27 局确定性回放、遥测汇总、双视口结果和三局正常时速发布实测。
- `R5高关战区与长期成长策划.md`：R5 八段战区、敌我成长闭环、行为/词缀/Boss、经济和双端规格。

- `远期总策划.md`：远期完整游戏总策划，包含产品定位、核心循环、战斗、关卡、角色、怪物、Boss、奖励、经济、任务、解锁和开发优先级。
