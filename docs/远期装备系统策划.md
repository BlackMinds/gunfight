# 装备系统策划文档

> 文档定位：本文是远期装备系统策划。当前原型已经以“配件实例 + 主词条/副词条 + 强化/重铸 + 背包整理”为核心，具体实现以 `配件与词条系统策划.md` 和 `当前原型策划.md` 为准。

> 2026-07-15 范围校正：本文件中的多武器、传说/神话、套装和元素系统都是候选方案，不属于当前重构承诺。当前先完成 8 槽配件的无损背包、准确命名、可解释比较和资源闭环。

## 1. 系统定位

装备系统是本游戏最核心的长期养成系统。玩家通过刷取武器、配件、词条和材料，形成不同战斗流派，持续提升推关能力。

装备系统由 4 层组成：

1. 武器：决定攻击方式。
2. 配件：改变武器行为和战斗节奏。
3. 词条：提供长期刷取目标。
4. 套装和特殊效果：形成后期构筑差异。

## 2. 装备部位

### 2.1 武器

玩家同一时间装备 1 把主武器。后期可开放副武器或支援武器。

武器包含：

- 武器类型
- 品质
- 等级
- 星级
- 基础属性
- 固定特性
- 随机词条
- 配件槽位

### 2.2 配件槽位

每把武器最多拥有 8 类配件槽位：

| 槽位 | 核心作用 |
| --- | --- |
| 枪口 | 后坐力、暴击、击退、扩散、消音 |
| 弹匣 | 弹量、换弹、射速、弹药循环 |
| 瞄具 | 射程、命中、暴击、索敌 |
| 枪管 | 射程、穿透、伤害、弹道 |
| 枪托 | 移动射击、稳定性、生存机动 |
| 弹芯 | 子弹类型、元素伤害、异常状态 |
| 模块 | 特殊机制、触发效果、主动或被动能力 |
| 芯片 | 经济、掉落、经验、流派强化、自动化逻辑 |

不同武器的槽位数量不同。前期武器可拥有 3 到 5 个槽位，后期高品质武器可拥有 6 到 8 个槽位。

当前原型规则：

- R1 重构前的原型允许自由选关；正式模式将限制到最高通关 +1，开发环境保留带风险标记的调试选关。
- 背包初始为空，配件需要通过关卡掉落进入背包。
- 普通关有概率掉落配件；每 5 关至少掉落 1 件配件，掉落结果会在结算奖励中展示。
- 战斗中配件背包锁定，不能临场换装；在通关结算页或基地中，背包点击配件即可装备。
- 若同槽位已有配件，会自动替换并把旧配件放回背包。
- 已装备配件、背包配件、角色等级、关卡和资源会写入本地存档。
- 当前配件已接入实例编号、roll 值、主词条、副词条、强化等级、强化和重铸。
- 当前不再提供基地永久强化入口；旧存档字段迁移为 0，不再形成隐藏战力。
- 当前仍未接入升星、套装效果和高级词条锁定。
- 当前基础背包容量为 24，禁止静默截断；溢出处理结果必须进入结算明细。

## 3. 装备品质

| 品质 | 颜色 | 词条数量 | 特殊效果 |
| --- | --- | --- | --- |
| 普通 | 白色 | 0-1 | 无 |
| 精良 | 绿色 | 1-2 | 低概率基础特效 |
| 稀有 | 蓝色 | 2-3 | 可出现流派词条 |
| 史诗 | 紫色 | 3-4 | 可出现强力触发效果 |
| 传说 | 橙色 | 4-5 | 固定传奇效果 |
| 神话 | 红色 | 5-6 | 神话效果或机制改变 |

品质影响：

- 基础属性范围
- 词条数量
- 词条数值上限
- 是否出现特殊效果
- 强化等级上限

## 4. 装备等级和强化

### 4.1 等级

装备通过消耗材料提升等级。

推荐等级上限：

| 品质 | 等级上限 |
| --- | --- |
| 普通 | 20 |
| 精良 | 40 |
| 稀有 | 60 |
| 史诗 | 80 |
| 传说 | 100 |
| 神话 | 120 |

### 4.2 升星

升星需要消耗同类装备或专用核心。

升星效果：

- 提升基础属性百分比
- 提升固定特性强度
- 解锁额外词条位
- 解锁特殊外观或弹道效果

### 4.3 重铸

重铸用于刷新随机词条。

规则建议：

- 可锁定 1 到 2 条词条。
- 锁定词条会提高重铸消耗。
- 特殊词条重铸概率较低。
- 神话效果不可普通重铸，只能通过高级材料洗练。

## 16. 流派构筑

### 16.1 暴击流

核心：

- 猎头枪口
- 狙击镜
- 弱点芯片
- 爆破模块

特点：

- Boss 输出高
- 依赖暴击率和暴击伤害
- 对怪群需要范围配件补足

### 16.2 射速触发流

核心：

- 快速弹匣
- 量子弹匣
- 电磁弹芯
- 连锁模块

特点：

- 高频触发异常和模块效果
- 清怪强
- 单发伤害低，需要后期词条支撑

### 16.3 穿透直线流

核心：

- 长枪管
- 穿甲枪管
- 穿甲弹芯
- 轨道枪管

特点：

- 对密集怪群强
- 对护甲敌人强
- 需要走位把敌人拉成直线

### 16.4 燃烧扩散流

核心：

- 燃烧弹芯
- 裂变弹芯
- 赤焰清道夫套装
- 黑洞模块

特点：

- 持续范围伤害
- 清怪效率高
- 对高抗性敌人需要破抗词条

### 16.5 生存吸血流

核心：

- 吸血模块
- 护盾模块
- 相位枪托
- 血色幸存者套装

特点：

- 容错高
- 适合挂机和高压关
- 输出上限低于纯伤害流

### 16.6 金币后勤流

核心：

- 悬赏芯片
- 幸运芯片
- 财阀芯片
- 黄金后勤套装

特点：

- 刷资源效率高
- 推极限关能力较弱
- 适合回刷和日常收益

## 19. 数值控制原则

### 19.1 避免单一最优解

每个流派都要有强项和短板。

- 暴击流强 Boss，但怕怪海。
- 燃烧流强清怪，但怕高抗性。
- 吸血流稳，但输出慢。
- 金币流收益高，但推关弱。

### 19.2 控制触发叠加

高频武器容易滥用触发效果，需要设置：

- 内置冷却
- 触发衰减
- 单目标触发上限
- 每秒触发上限

### 19.3 装备替换节奏

前期频繁替换装备，中期追求词条组合，后期追求毕业品质和套装。

| 阶段 | 装备目标 |
| --- | --- |
| 1-200 | 凑齐部位 |
| 201-1000 | 找到适合武器 |
| 1001-3000 | 形成流派 |
| 3001-7000 | 优化词条和套装 |
| 7001-10000 | 追求神话和毕业构筑 |

## 20. 分阶段落地顺序

1. R1：24 格无损背包、生命模块迁移、整数穿透规则、三维比较。
2. R2：金币重铸手续费、合金锁词条、收藏保护、自动回收规则。
3. R3：在第 1～100 关验证穿透、火力、机动三类构筑，不新增品质层。
4. R4：确认武器形态需求后，再实现多武器和元素弹芯。
5. 套装、传说、神话必须在对应关卡阶段立项，不提前进入掉落池。

任何新装备系统都要回答三个问题：它改变什么战斗决策、克制什么关卡压力、使用什么资源维护。只增加更大的百分比不算新系统。
