# R4 与 R5 真人验收记录表

## 1. 使用规则

- 本表只填写真人从开局到结算全程操作的有效局；Codex 浏览器连续控制、自动回放、布局截图和脚本输入均不得填写为“真人通过”。
- 每局开始前记录提交号、设备、分辨率和输入方式。崩溃、失焦中断、错用构筑或非本人全程操作的样本标记为“无效需重跑”，不能补写估算结果。
- R4 的第 350、450、500 关各至少完成 1 局；R5 按表内指定设备完成 6 局。
- 单局结论只填写“通过 / 不通过 / 无效需重跑”；发现阻断问题时先记录并停止推进，修复后从该节点重跑。

1. R4：350 → 450 → 500，先补齐较早版本的三局真人门槛。
