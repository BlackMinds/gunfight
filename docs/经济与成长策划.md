# 经济与成长策划

## 1. 当前经济目标

当前版本经济系统服务两个目标：

1. 支撑关卡推进。
2. 支撑配件筛选、强化、重铸和背包整理。

当前版本不再提供基地永久强化卡入口。

2026-07-15 重构目标是先建立最小经济闭环和无损背包规则。金币、合金若尚未具备实际消耗口，界面必须明确其“储备资源”状态，不能把数值增长等同于有效成长。

## 2. 当前资源

| 资源 | 当前来源 | 当前用途 |
| --- | --- | --- |
| 金币 | 战斗拾取、关卡结算、拆解零件、清理配件 | R2 起作为重铸手续费和背包扩容货币。 |
| 经验 | 战斗拾取、关卡奖励 | 提升角色等级，增加生命成长。 |
| 合金 | 关卡结算 | 重铸时锁定 1 条副词条的额外消耗。 |
| 零件 | 关卡结算、清理配件 | 配件强化、配件重铸、拆解成金币。 |
| 配件 | 关卡掉落 | 装备、比较、强化、重铸、清理。 |

## 3. 角色成长

当前角色成长主要来自：

- 角色等级。
- 当前装备配件。
- 配件词条。

角色升级当前只提高最大生命。该成长不足以支撑 100～10000 关的敌人生命与伤害倍率，因此在伤害成长、装备梯度和关卡内容完成前，不把高关编号视为可玩成长线。

## 4. 已移除的基地永久强化入口

此前原型中存在以下基地强化卡：

- 枪械校准。
- 护甲整备。
- 磁吸轨道。

这些入口已经从当前页面删除。

旧存档中可能仍保留 `weaponLevel`、`armorLevel`、`magnetLevel` 字段。R1 存档迁移会保留字段结构但把数值归零，新战斗不再读取这些等级作为隐藏战力。迁移后写入 `saveVersion`，避免同样装备因为存档年代不同而得到不同属性。

## 5. 当前成长循环

当前成长循环如下：

1. 选择目标关卡。
2. 战斗获得资源和配件。
3. 结算页查看掉落。
4. 在基地整理背包。
5. 比较同槽配件。
6. 装备更合适的配件。
7. 对好底子配件强化或重铸。
8. 再挑战更高关卡。

## 6. 资源消耗设计

### 6.1 配件强化

强化消耗零件。

设计意图：

- 让玩家把资源投入到具体实例，而不是抽象属性。
- 让高品质、高 roll、好副词条的配件更值得保留。

### 6.2 配件重铸

重铸同时消耗零件和金币；锁定 1 条副词条时额外消耗合金。

| 品质 | 零件 | 金币 | 锁定副词条合金 |
| --- | ---: | ---: | ---: |
| 普通 | 2 | 40 | 1 |
| 精良 | 5 | 70 | 1 |
| 稀有 | 7 | 110 | 2 |
| 史诗 | 10 | 160 | 3 |

零件成本按当前副词条数量计算；表中为各品质标准副词条数量下的实际成本。资源在执行重铸时一次性校验和扣除，任一资源不足都不改变配件或部分扣款。

设计意图：

- 刷新副词条，保留主词条和强化等级。
- 让“刷到好底子”有后续追求。
- 金币成为可重复消耗口，合金用于高价值定向优化。

### 6.3 清理低品质

清理低品质配件返还金币和零件。

设计意图：

- 降低背包噪音。
- 给低价值掉落提供回收价值。

### 6.4 背包容量与溢出

- R1 容量统一为 24 件，并显示 `已用 / 24`。
- 获得掉落时先写入背包，再统一检查容量。
- 超出容量的最低优先级配件自动回收为金币和零件，并在结算列出名称与返还；收藏、已装备和本次高品质掉落不能被自动回收。
- 如果自动回收尚未完成，则必须阻止继续部署并要求玩家整理，不能使用 `.slice()` 静默截断。
- 收藏或本次掉落等受保护实例超过 24 件时，背包允许暂时超容并阻止部署，不得为了恢复到 24 件而破坏保护。

## 7. 当前经济风险

| 风险 | 表现 | 后续处理 |
| --- | --- | --- |
| 金币产消比待验证 | 重铸已形成稳定消耗，但高关收入尚无连续样本 | 在第 20/40/60/100 关固定构筑试玩时记录每关可重铸次数。 |
| 合金产消比待验证 | 锁词条已形成消耗，但高品质连续重铸成本较高 | 记录无锁/有锁重铸占比和资源卡点。 |
| 零件过度重要 | 强化和重铸都消耗零件 | 后续需要区分普通零件、重铸芯片或精密元件。 |
| 背包静默截断 | 当前容量检查使用数组截断 | R1 改为 24 容量与显式溢出回收。 |
| 隐形旧存档战力 | 已删除入口仍参与公式 | R1 迁移为 0 并写入存档版本。 |

## 8. 下一步经济建议

### 8.1 短期

- R1：背包无损、旧存档迁移、返还明细。
- R2：重铸加入金币消耗，锁定 1 条副词条消耗合金。（已完成）
- R2：背包容量可用金币从 24 扩展到 32，再扩展到 40。

### 8.2 中期

- 增加配件商店。
- 增加背包扩容。
- 增加每日资源关。

### 8.3 长期

- 资源分层：金币、零件、重铸芯片、精密元件。
- 高品质配件突破。
- 套装或流派养成消耗。

## 9. 经济验收指标

| 指标 | R1/R2 目标 |
| --- | --- |
| 静默物品损失 | 0 次 |
| 金币主要消耗 | 至少 1 个可重复消耗口 |
| 合金主要消耗 | 词条锁定落地后再评价产消比 |
| 零件用途占比 | 强化与重铸分开记录，避免单一资源堵死全部成长 |
| 背包整理频率 | 正常连打 3～5 关后再需要整理，而非每关强制中断 |
| 旧存档战力差 | 相同角色等级和装备下为 0 |
