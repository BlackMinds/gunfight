# 枪战放置 RPG 完整游戏策划文档

> 文档定位：本文是远期总策划，包含大量未落地设想。当前原型实现、已删除功能和短期迭代优先参考 `当前原型策划.md`、`战斗与关卡循环策划.md`、`配件与词条系统策划.md` 和 `经济与成长策划.md`。

> 范围校正记录：2026-07-15 曾将发布验证上限收敛到第 100 关，等待对应成长系统、敌人机制和连续试玩完成；2026-07-21 R4/R5 验收与发布实测通过后，正式范围已开放到第 10000 关。

## 1. 产品定位

### 1.1 暂定名称

Gunfight Idle RPG

### 1.2 游戏类型

枪战 / 放置 / RPG / 装备刷取 / 构筑养成 / 长线推关

### 1.3 核心卖点

玩家操控一名持枪角色在战场中移动，怪物会从四周不断出现并朝玩家追击。角色默认自动射击，玩家通过走位、武器选择、配件组合、装备词条和局外养成，不断突破更高关卡。

游戏长期内容容量为 1 到 10000 关。版本采用阶段门方式逐段交付，不能先开放全部编号、再依赖生命和伤害膨胀填充。每个阶段必须同时具备新压力、新成长解法和可复现的验收样本。

### 1.4 目标体验

- 战斗时爽快：大量怪物围攻，自动火力持续清场。
- 操作上轻量：玩家主要负责移动、走位、选择技能和配置装备。
- 成长上耐刷：武器、配件、词条、天赋和关卡奖励形成长期追求。
- 构筑上有深度：不同武器和配件组合会改变子弹行为、输出方式和生存能力。

## 2. 玩家画像

### 2.1 主要用户

- 喜欢放置游戏和长期养成的玩家
- 喜欢刷装备、刷词条、推图爬塔的玩家
- 喜欢枪械、弹幕、割草和自动战斗的玩家
- 喜欢低操作门槛但有构筑空间的玩家

### 2.2 游玩场景

- 碎片时间：打 3 到 5 关，收取奖励。
- 日常游玩：完成每日任务、强化装备、推进关卡。
- 长线目标：冲击 1000、3000、7000、10000 等阶段节点。

## 3. 游戏核心循环

### 3.1 战斗循环

1. 进入关卡。
2. 怪物从地图边缘刷新并追击玩家。
3. 玩家移动躲避包围。
4. 角色自动攻击最近或最有威胁的敌人。
5. 击杀敌人获得经验、金币、材料和掉落。
6. 击败全部波次或 Boss 后通关。
7. 进入结算，领取奖励。

### 3.2 养成循环

1. 推进关卡获得资源。
2. 强化角色等级、天赋、武器和配件。
3. 刷取更高品质装备和更优词条。
4. 调整流派构筑。
5. 挑战更高阶段关卡。

### 3.3 长线循环

1. 解锁新系统。
2. 形成新流派。
3. 击败阶段 Boss。
4. 解锁更高品质掉落池。
5. 进入下一阶段难度。

## 4. 基础战斗设计

### 4.1 操作方式

- 桌面端：WASD 或方向键移动。
- 移动端：虚拟摇杆移动。
- 攻击：默认自动攻击。
- 技能：中后期可加入主动技能按钮。
- 拾取：默认靠近自动拾取，拾取范围受属性影响。

### 4.2 战斗视角

推荐使用俯视角 2D 战斗。

角色位于屏幕中心附近，摄像机跟随角色移动。怪物从屏幕外或地图边缘刷新并向角色靠近。

### 4.3 自动攻击

默认目标规则：

| 优先级 | 规则 |
| --- | --- |
| 1 | 最近敌人 |
| 2 | 攻击范围内血量最低敌人 |
| 3 | 精英怪 |
| 4 | Boss |

可通过配件或芯片改变索敌逻辑，例如优先攻击 Boss、优先攻击精英、优先攻击残血敌人。

### 4.4 伤害类型

| 类型 | 说明 |
| --- | --- |
| 物理 | 基础枪械伤害，受护甲影响 |
| 爆炸 | 范围伤害，适合清怪 |
| 火焰 | 造成持续灼烧 |
| 电击 | 概率连锁多个敌人 |
| 毒素 | 长时间持续伤害，可叠层 |
| 冰霜 | 减速、冻结、控制 |
| 能量 | 部分无视护甲，适合打重装敌人 |

### 4.5 状态效果

| 状态 | 效果 |
| --- | --- |
| 灼烧 | 每秒受到火焰伤害 |
| 感电 | 被命中时可能传导伤害 |
| 中毒 | 持续掉血，可叠加 |
| 冰缓 | 移动速度降低 |
| 冻结 | 短时间无法移动 |
| 流血 | 移动时额外受到伤害 |
| 破甲 | 护甲降低 |
| 易伤 | 受到伤害提高 |
| 击退 | 被子弹推开 |
| 眩晕 | 短时间无法行动 |

## 5. 关卡系统

### 5.1 总关卡

长期目标关卡数量：10000；当前开发验证范围：1～100。

玩家每通关一关，解锁下一关。开发环境可以跳关测试，但正式存档不能通过自由选关绕过成长和奖励节奏。失败时保留已拾取收益，但无法领取完整通关奖励。

### 5.2 阶段划分

| 关卡 | 阶段名 | 核心变化 |
| --- | --- | --- |
| 1-50 | 训练区 | 教学、基础敌人、基础武器 |
| 51-200 | 街区战 | 快速怪、重装怪、基础配件 |
| 201-500 | 废城推进 | 精英怪、异常状态、配件强化 |
| 501-1000 | 军工封锁 | Boss 机制、稀有装备、天赋树 |
| 1001-2000 | 荒原战线 | 词缀怪、元素流派、离线收益 |
| 2001-3000 | 合金要塞 | 护甲体系、穿透体系、史诗装备 |
| 3001-5000 | 辐射城区 | 高密度怪群、异常抗性、传说装备 |
| 5001-7000 | 深层战场 | 多阶段 Boss、组合词缀、套装 |
| 7001-9000 | 黑域行动 | 神话装备、极限构筑、特殊挑战 |
| 9001-10000 | 终局战争 | 全系统检验、终局 Boss、毕业追求 |

### 5.3 关卡类型

| 类型 | 出现规则 | 说明 |
| --- | --- | --- |
| 普通关 | 大部分关卡 | 多波怪物，节奏稳定 |
| 精英关 | 每 5 关 | 精英怪数量提升 |
| Boss 关 | 每 10 关 | 出现 Boss |
| 大 Boss 关 | 每 100 关 | 阶段性检验 |
| 资源关 | 每 25 关 | 材料和金币奖励更高 |
| 生存关 | 特定节点 | 限时存活 |
| 悬赏关 | 中后期解锁 | 指定击杀目标 |

### 5.4 难度公式

关卡基础倍率：

```ts
hpMultiplier = 1 + level * 0.08 + Math.pow(level, 1.18) * 0.015
damageMultiplier = 1 + level * 0.035 + Math.pow(level, 1.08) * 0.004
speedMultiplier = 1 + Math.min(level / 3000, 1.5)
rewardMultiplier = 1 + level * 0.06 + Math.pow(level, 1.12) * 0.01
```

敌人最终属性：

```ts
enemyHp = baseHp * hpMultiplier * typeHpRate * stageRate
enemyDamage = baseDamage * damageMultiplier * typeDamageRate * stageRate
enemySpeed = baseSpeed * speedMultiplier * typeSpeedRate
```

现公式只作为原型基线，不能直接外推到 10000 关。第 100 关以后必须改为分段曲线：伤害和移速使用软上限，生命成长与玩家有效 DPS 成长成对验证；每 1000 关的离散 `stageRate` 跳变需要改成平滑阶段系数或明确阶段门。

### 5.5 难度节奏控制

为了避免 10000 关只变成纯数值膨胀，每 1000 关加入新的压力来源：

| 节点 | 新增压力 |
| --- | --- |
| 1000 | 精英怪词缀变多 |
| 2000 | 护甲敌人比例提高 |
| 3000 | 异常抗性出现 |
| 4000 | Boss 进入双技能组合 |
| 5000 | 怪物刷新密度提升 |
| 6000 | 敌人获得阵营光环 |
| 7000 | 部分敌人具备远程压制 |
| 8000 | 精英怪双词缀 |
| 9000 | Boss 多阶段形态 |
| 10000 | 终局 Boss |

每个阶段开放前必须满足：至少 2 种新敌人行为或组合、至少 1 种相应成长解法、阶段 Boss、经济消耗口、固定构筑连续 3 局通过和至少 1 次真人预警辨识验收。

## 6. 角色系统

### 6.1 基础属性

| 属性 | 作用 |
| --- | --- |
| 生命值 | 决定承伤能力 |
| 攻击力 | 所有武器的基础伤害来源 |
| 防御 | 减少受到的直接伤害 |
| 护甲 | 抵消物理伤害 |
| 移动速度 | 决定走位和拉扯能力 |
| 射速 | 影响开火频率 |
| 暴击率 | 触发暴击的概率 |
| 暴击伤害 | 暴击时的伤害倍率 |
| 换弹速度 | 降低换弹耗时 |
| 弹匣容量 | 提升连续输出时间 |
| 穿透 | 子弹可穿过敌人的数量 |
| 击退 | 命中后推开敌人 |
| 吸血 | 按伤害比例恢复生命 |
| 生命恢复 | 每秒自然恢复 |
| 拾取范围 | 自动吸取掉落物范围 |
| 幸运 | 影响掉落品质和词条质量 |
| 冷却缩减 | 降低技能和模块触发间隔 |

### 6.2 角色等级

角色等级通过通关、击杀和离线收益获得经验。

角色升级提供：

- 基础生命值提升
- 基础攻击力提升
- 天赋点解锁
- 部分系统解锁条件

### 6.3 天赋树

天赋树分为 5 个方向：

| 分支 | 定位 |
| --- | --- |
| 火力 | 攻击、暴击、射速、穿透 |
| 生存 | 生命、防御、护甲、恢复 |
| 机动 | 移速、闪避、拾取范围 |
| 工程 | 配件强化、模块冷却、异常效果 |
| 后勤 | 金币、经验、掉落、离线收益 |

## 7. 武器系统

武器决定主要攻击形态，配件决定构筑方向。完整装备细节见 `docs/远期装备系统策划.md`。

### 7.1 武器类型

| 武器 | 定位 | 优势 | 短板 |
| --- | --- | --- | --- |
| 手枪 | 均衡入门 | 稳定、易成型 | 上限较低 |
| 双枪 | 高频移动 | 移动灵活、射速快 | 单发伤害低 |
| 冲锋枪 | 触发特效 | 高频命中 | 依赖弹药和换弹 |
| 霰弹枪 | 近距离爆发 | 多弹丸、击退强 | 远距离弱 |
| 突击步枪 | 泛用主力 | 属性均衡 | 缺少极端优势 |
| 狙击枪 | 单体爆发 | 高暴击、高穿透 | 射速慢 |
| 轻机枪 | 持续压制 | 弹量大、清怪稳 | 换弹慢、机动差 |
| 榴弹发射器 | 范围清怪 | 爆炸范围伤害 | 子弹速度慢 |
| 火焰喷射器 | 持续灼烧 | 面积压制 | 射程短 |
| 电磁枪 | 连锁伤害 | 群体传导 | 对单体不稳定 |
| 激光枪 | 持续锁定 | 命中稳定 | 需要持续瞄准 |
| 等离子炮 | 后期重武器 | 高伤害、能量伤 | 射速慢、成本高 |

### 7.2 武器成长

武器拥有：

- 等级
- 品质
- 星级
- 基础属性
- 固定特性
- 随机词条
- 配件槽位

武器成长方式：

- 消耗武器经验升级
- 消耗同名武器或核心材料升星
- 消耗材料突破等级上限
- 后期可重铸部分词条

### 12.3 离线收益

离线收益根据最高通关关卡计算。

建议规则：

- 默认最多累计 8 小时。
- 天赋和芯片可提升累计上限。
- 离线收益主要产出金币、经验、少量材料。
- 高品质装备不直接通过离线收益大量产出，避免削弱主动游玩价值。

## 13. 任务和成就

### 13.1 每日任务

- 通关指定数量关卡
- 击杀指定数量敌人
- 击败 Boss
- 强化装备
- 完成一次资源关

### 13.2 周常任务

- 推进指定关卡数
- 击败多个 Boss
- 获得指定品质装备
- 完成高危节点；独立挑战玩法上线后再替换为挑战任务

### 13.3 成就

- 首次通关第 100 关
- 首次获得传说装备

## 15. 商业化预留

如果后续需要商业化，优先选择不破坏核心战斗公平性的内容。

可选方向：

- 外观皮肤
- 通行证
- 月卡
- 离线收益加速
- 资源礼包
- 仓库扩容
- 重铸材料

需避免：

- 直接出售毕业装备
- 强制广告卡进度
- 过度依赖抽卡推进主线

## 16. 完整版本开发优先级

开发不按“先堆到 10000 关”推进，而按可封板区间推进：

| 版本 | 关卡范围 | 必须完成 |
| --- | --- | --- |
| R1/R2 | 1～10 | 数据可信、背包安全、基地与战斗界面重构 |
| R3 | 1～100 | 四个新验证节点、正式关卡锁、资源闭环 |
| R4 | 101～500 | 新敌人机制、伤害成长、精英词缀首版 |
| 后续 | 每阶段单独立项 | 不继承未经验证的全局指数曲线 |
